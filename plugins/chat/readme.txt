ОПИСАНИЕ
Очередной плагин чата для livestreet. Данный плагин реализует чат на сайте в popup окне. Идейный вдохновитель данной реализации сайт http://www.wallpapers.ru/ .
Демо можно посмотреть залогинившись на сайте http://netlanc.net. Бесплатная версия.

Обязательным условием использования плагина является наличие активной ссылки на сайт разработчика Netlanc (http://catalog.netlanc.net/),
которая уже прописана в плагине. Отключить ссылку можно за донейт от 5уе, реквизиты для донейта в пункте ВАШЕ УЧАСТИЕ.

УСТАНОВКА
1. Скопировать плагин в каталог /plugins/
2. Через панель управления плагинами (/admin/plugins/) запустить его активацию
3. Выполнить настройки плагина в файле /plugins/chat/config/config.php
4. Радоваться ;)
