ОПИСАНИЕ
Плагин для поиска и очистки неиспользуемых изображений, финального удаления коментариев и очистки и или востановления неиспользуемых связей

Обязательным условием использования плагина является наличие активной ссылки на сайт разработчика Netlanc (http://netlanc.net/),
которая уже прописана в плагине. Отключить ссылку можно за донейт от 5уе.

УСТАНОВКА
1. Скопировать плагин в каталог /plugins/
2. Через панель управления плагинами (/admin/plugins/) запустить его активацию.
3. "Пылесосить" перейдя по адресу http://<ваш_сайт>/admin/cleaner/
4. При желании поставить в cron файл plugins/cleaner/include/cron/cron.php ( например 0 * * * * /usr/bin/wget -O - -q _полный_серверный_путь_к_файлу_/plugins/cleaner/include/cron/cron.php)
5. Радоваться ;)
